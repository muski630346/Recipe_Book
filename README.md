# Recipe Book Web Application

## Project Overview

The Recipe Book Web Application is a multi-page frontend project that allows users to browse and view different recipes with detailed ingredients and preparation steps.

The application includes:
- User Login Interface
- Recipe Listing Page
- Detailed Recipe View Page
- Ingredients and Preparation Sections
- Tags and Recipe Metadata Display
- Clean Green Themed User Interface

---

## Features

### 1. Authentication Page
- Email and Password Login Form
- Google Login Button (UI only)
- Apple Login Button (UI only)
- Forgot Password Option
- Navigation to Intro Page

### 2. Recipe Listing Page
- Displays multiple recipes
- Recipe Image Thumbnail
- Short Ingredient Description
- View Recipe Button
- Card-style Layout Design

### 3. Recipe Detail Page
- Large Recipe Image
- About Section
- Tags Section
- Yield Information
- Creator and Date Added
- Ingredients List
- Step-by-step Preparation Guide
